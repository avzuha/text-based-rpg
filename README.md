# text-based-rpg
A simple text-based RPG written in **C++**.  
Players can choose between **Warrior** or **Mage**, battle enemies, use items, and experience random events while leveling up through multiple rounds.

---

## Features
- **Character Classes**: Warrior (high HP & defense, Power Strike) or Mage (high attack, Fireball).
- **Turn-Based Combat**: Normal attacks, special attacks, item usage, or fleeing.
- **Inventory System**: Potions, weapons, and armor with equip/use mechanics.
- **Leveling System**: Gain XP, level up, and improve stats.
- **Random Events**: Traps, merchants, healers, and treasures.
- **Enemy Variety**: Goblins, Skeletons, Trolls, Dark Mages, and Dragon Whelps with loot drops.

---

## Project Structure
- rpg.cpp → Main source code containing all game logic.
- Classes:
- Character (base class)
- Warrior and Mage (player classes)
- Enemy (various enemy types)
- Item (inventory system)
